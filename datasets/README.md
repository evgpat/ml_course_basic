# Данные

## `wineQualityReds.csv`
Input variables (based on physicochemical tests):  
* `fixed acidity`
* `volatile acidity`
* `citric acid`
* `residual sugar`
* `chlorides`
* `free sulfur dioxide`
* `total sulfur dioxide`
* `density`
* `pH`
* `sulphates`
* `alcohol`

Output variable (based on sensory data):
* `quality` (score between 0 and 10)
